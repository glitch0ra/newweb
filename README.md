# Медиа-галерея

Одностраничное веб-приложение (SPA) для демонстрации авторских медиа-работ.

## Структура проекта

```
newweb/
├── index.html          # Главный HTML файл
├── css/
│   └── styles.css      # Все стили (Liquid Glass + Anti-Design)
├── js/
│   ├── router.js      # Роутинг SPA
│   ├── data-loader.js  # Загрузка JSON данных
│   ├── app.js         # Главный файл приложения
│   ├── components/
│   │   ├── modal.js   # Модальные окна
│   │   └── carousel.js # Карусели
│   └── pages/
│       ├── main.js        # Вкладка "Главная"
│       ├── collections.js # Вкладка "Коллекции"
│       ├── screenshots.js # Вкладка "Скриншоты"
│       ├── videos.js      # Вкладка "Видео"
│       ├── history.js     # Вкладка "История"
│       └── about.js        # Вкладка "Об авторе"
├── data/
│   ├── main.json       # Данные для главной страницы
│   ├── collections.json # Данные коллекций
│   ├── screenshots.json # Данные скриншотов
│   ├── videos.json     # Данные видео
│   ├── history.json    # Данные истории
│   └── about.json      # Данные об авторе
└── assets/
    ├── background.webm # Фоновое видео
    └── header.webm     # Видео для шапки
```

## Технологии

- HTML5
- Vanilla JavaScript (ES6+)
- CSS3 (Flexbox, Grid, Custom Properties, Animations)
- JSON для данных

## Особенности

- **Liquid Glass дизайн**: Прозрачные элементы с размытием фона
- **Anti-Design**: Асимметрия, glitch-эффекты, визуальный шум
- **SPA навигация**: Переключение между вкладками без перезагрузки
- **Модальные окна**: Полноэкранный просмотр медиа-контента
- **Карусели**: Горизонтальная прокрутка с drag/swipe поддержкой
- **Ленивая загрузка**: Оптимизация загрузки изображений
- **Адаптивный дизайн**: Поддержка mobile, tablet, desktop

## Установка

1. Клонируйте репозиторий
2. Добавьте видео файлы в папку `assets/`:
   - `background.webm` - фоновое видео
   - `header.webm` - видео для шапки
3. Обновите JSON файлы в папке `data/` своими данными
4. Откройте `index.html` в браузере

## Использование

- Навигация между вкладками через меню в шапке
- Клик по любому медиа-блоку открывает модальное окно
- В модальных окнах можно перелистывать контент стрелками или клавишами ← →
- ESC или клик вне модального окна закрывает его
